## 4. 주요 기능

# HealthTube

/ -> home 화면
/join -> 회원가입
/login -> 로그인
/search -> 검색기능

/users/:id -> See User
/users/logout -> 로그 아웃
/users/edit -> 사용자 편집
/users/delete -> 사용자 삭제

/videos/:id -> 영상보기
/videos/watch -> 영상 시청
/video/:id/edit -> 영상 편집
/video/:id/delete -> 영상 삭제
/video/upload -> 영상 업로드

/video/comments -> 영상 댓글
/video/comments/delete -> 영상 댓글 삭제
